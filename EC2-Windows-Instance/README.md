# AWS EC2 Windows Instance

## 📌 Project Overview
This project demonstrates deployment and configuration of a Windows EC2 instance in AWS.

## 🚀 Services Used
- Amazon EC2
- VPC
- Security Groups
- RDP

## ⚙️ Tasks Performed
- Created Windows EC2 instance
- Configured Security Group
- Enabled RDP access
- Connected remotely to Windows Server
